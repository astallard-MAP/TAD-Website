# The Auction Department (TAD) Website 🐸

A state-of-the-art marketing platform and member portal for the UK Land and Property Auction sector.

## 🚀 Development Stack
- **Framework**: [Next.js](https://nextjs.org/) (App Router, Version 14+)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: Vanilla [CSS Modules](https://github.com/css-modules/css-modules)
- **Branding**: The Auction Department Limited (Minimalistic, Premium)
- **Time/Date**: EN UK Standard (dd/mm/yyyy, GMT/BST)

## 🐸 Brand Ambassador: Frank Tadsworth-Bids
Meet Frank, our Chief AI Auctioneer. Frank is a common frog in a sharp purple suit, always ready with his gavel. He embodies our "look before you leap" philosophy, providing expert guidance through the auction process.

### 🖼️ Required Assets (Place in /public/images/)
- `logo.png`: Main brand logo.
- `face.png`: Frank's headshot for AI chat.
- `FTB.png`: Frank's full-body shot for the About page.

## 📄 Key Pages
- **Marketing**: Home, Properties, Sell, Buy, Valuation, Instant Offer, Blog.
- **Member Portal**: Login, Register, Profile (User Dashboard).
- **Communication**: Contact Us, Auction Dates Calendar.
- **Legal**: Privacy Policy, Cookie Policy, Complaints Procedure, Site Map.

## 🛠️ Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Access the Site**: Open [http://localhost:3000](http://localhost:3000)
